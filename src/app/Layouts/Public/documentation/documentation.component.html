Background:

Api can be accessed at: https://masterofpuppets.herokuapp.com/

The Postman collection holds 2 variables:

DevEndpoint : refers to the local development endpoint

LiveEndpoint : refers to the public api

A FeatureToggle Object consists of the following values:

{{ '{' }}

secretKey: {{ '{' }} type: String, required: true, unique: true},

toggles: {{ '{' }} type: [ {{ '{' }} id: String, state: Boolean} ], required: true }

}

secretKey: Your unique identifier. This identifies a featuretoggle, and allows you to target a specific FeatureToggle for operations such as CRUD.

toggles[]: An array of individual toggles belonging to the specified secretKey. Each toggle simply consists of an identifier and a boolean belonging to said identifier.
